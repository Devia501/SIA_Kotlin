## Panduan Singkat

1. Clone repository ini:
   ```bash
   git clone https://github.com/Devia501/SIA_Kotlin.git
   ```
2. Masuk ke branch masing-masing:
   ```bash
   git checkout branch-bagianKalian
   ```
3. Kerjakan bagianmu dan push perubahan:
   ```bash
   git add .
   git commit -m "update fitur kamu"
   git push origin branch-bagianKalian
   ```
